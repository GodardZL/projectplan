# 竞品分析

## 目录说明
本目录包含项目计划管理产品的竞品分析资料。

## 文件结构
- `竞品概览.md` - 主要竞品产品概览
- `功能对比分析/` - 详细功能对比
  - `核心功能对比.md` - 项目管理核心功能对比
  - `用户体验对比.md` - 用户界面和体验对比
  - `定价策略对比.md` - 各竞品定价策略分析
- `竞品深度分析/` - 单个竞品深度分析
  - `Asana分析.md`
  - `Trello分析.md`
  - `Monday.com分析.md`
  - `Notion分析.md`
  - `飞书项目分析.md`
- `SWOT分析.md` - 竞品优劣势分析
- `市场定位分析.md` - 竞品市场定位分析

## 分析维度
- 产品功能
- 用户体验
- 技术架构
- 商业模式
- 市场表现
