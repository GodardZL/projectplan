# 原型设计

## 目录说明
本目录包含产品原型设计相关资料。

## 文件结构
- `设计规范/` - 设计规范和指南
  - `UI设计规范.md` - 用户界面设计规范
  - `交互设计规范.md` - 交互设计规范
  - `色彩和字体规范.md` - 视觉设计规范
- `线框图/` - 页面线框图
- `高保真原型/` - 高保真交互原型
- `用户流程图/` - 用户操作流程图
- `信息架构图.md` - 产品信息架构
- `设计评审记录/` - 设计评审会议记录

## 设计工具
- Figma - 主要设计工具
- Sketch - 备选设计工具
- Axure - 交互原型工具
- Miro - 流程图和架构图

## 设计原则
- 用户中心设计
- 简洁易用
- 一致性
- 可访问性
